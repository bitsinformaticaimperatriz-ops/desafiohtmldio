O que é HTML?

Voltar

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web. Ela define a estrutura e o conteúdo de uma página web usando elementos como títulos, parágrafos, links, imagens e formulários. Ela foi criada por Tim Berners-Lee em 1991 e desde então tem evoluído para suportar novas funcionalidades e melhorar a experiência do usuário na web.

Estrutura de uma página HTML

Voltar

A estrutura básica de uma página HTML inclui a declaração do tipo de documento (DOCTYPE html), a tag (html) que envolve todo o conteúdo, a seção head que contém metadados e links para recursos externos, e a seção body que contém o conteúdo visível da página, como texto, imagens e links.

Diferença entre HTML e HTML5

Voltar

HTML5 é a versão mais recente do HTML, introduzida em 2014. Ele traz várias melhorias e novas funcionalidades em relação às versões anteriores do HTML, como suporte nativo para áudio e vídeo, elementos semânticos aprimorados como: (article, section e nav), APIs para armazenamento local e geolocalização, além de melhor suporte para dispositivos móveis e acessibilidade.

Tags e Atributos

Voltar

Tags são os elementos básicos do HTML que definem a estrutura e o conteúdo de uma página web, enquanto atributos são propriedades adicionais que fornecem informações extras sobre as tags. Por exemplo, a tag (a) é usada para criar links, e o atributo "href" especifica o destino do link.

Formulários

Voltar

Formulários em HTML são usados para coletar dados dos usuários. Eles consistem em vários elementos, como campos de texto, botões de rádio, caixas de seleção e botões de envio. Os formulários são definidos pela tag form e podem incluir atributos como "action" (para especificar o destino dos dados) e "method" (para definir o método de envio, como GET ou POST).

Elementos Semânticos

Voltar

Elementos semânticos em HTML5 são tags que descrevem claramente o significado do conteúdo que elas envolvem. Exemplos incluem header (cabeçalho), footer (rodapé), article (artigo), section (seção) e nav (navegação). Esses elementos ajudam a melhorar a acessibilidade, SEO e a estrutura geral do documento.
